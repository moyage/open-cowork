# Requirements

- Project entry text must state that open-cowork is a project-level collaboration method, not a required platform Skill name.
- Generated Agent entry files must tell Claude Code, Codex, and other Agents how to continue when a platform Skill is unavailable.
- Execution readiness must require human-confirmed collaborators, external rule selection or explicit skip, and Step 1-4 artifacts.
- Status output must show missing readiness items in a human-readable way.
- Smoke verification must not fail only because the repository currently has no `tests/` directory.
